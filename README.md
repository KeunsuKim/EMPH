# EMPH

This repository contains the implementation of the Exact Multi-Parameter Persistent Homology (EMPH) method proposed in our paper:

Keunsu Kim and Jae-Hun Jung,
Exact Multi-Parameter Persistent Homology of Time-Series Data: Fast and Variable Topological Inferences


## Datasets

The datasets used in this study are publicly available from the following sources:

Example 1: Four Shapes Dataset
Available on Kaggle: https://www.kaggle.com/datasets/smeschke/four-shapes

Example 2: ProximalPhalanxOutlineAgeGroup Dataset
Available from the UCR Time Series Classification Archive:
https://www.cs.ucr.edu/~eamonn/time_series_data_2018/

Example 3: Resting-State fMRI Dataset
Available on GitHub: https://github.com/laplcebeltrami/rsfMRI
